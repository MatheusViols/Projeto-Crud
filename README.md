# Projeto-Crud

Usar:
    pip install mysql-connector-python
Para instalar o pacote e conseguir importar mysql.connector

A não ser em casos de teste especifico, para executar sempre use:
    python main.py

Os scripts de criação do usuario, banco e tabela estão em script-banco.
Uma versão para linux e outra para windows. Tente executar como administrador.
